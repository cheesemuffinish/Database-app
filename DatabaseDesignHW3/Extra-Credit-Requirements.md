Set of requirements for developing our application

Tools used:
Microsoft SQL Server
Node.JS
Bootstrap
Express.JS
NPM package manager

Packages to install:
1.) Sudo apt-get update your system
2.) Install Npm - This will install all app dependencies:
	- Tediuous
	- Express
	- Morgan
	- Pug
	- Debug
3.) Install Node - needed for backend and server (express) development


Back End Requirements:
- Establish a connection to SQL Server
- Create API for each service 
- Ensure each "/" endpoint routes to specific service
- create route for each endpoint


Front End Requirements:
- Create credentials to log into app and query
- Create HTML form inputs for each of the queries
- Create javascript to make API requests
- Ensure the results from the query satisfy the actual results
